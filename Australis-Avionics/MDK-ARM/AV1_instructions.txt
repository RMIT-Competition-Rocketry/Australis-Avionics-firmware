To erase firmware on A5 boards.
1. Open JLink Commander
2. For Supervisor: IDPre:0, DRPre = 0
3. For Main: IDPre:9, DRPre = 2
4. halt
5. erase
6. Close Commander 
7. Power cycle
8. Repeat process for other core.

To load main CPU core.
1. Ensure that 'GM Primary' is uncommmented in JLinkSettings.JLinkScript
2. Build Code.
3. Press the 'Load Button' in Keil uVision
4. Power Cycle the CPU (code will run immediately).
5. Enter Debug Session by Pressing Ctrl+F5
6. Press 'Stop'
7. Set Program Counter (PC) to 0x0800_0000
8. Press F5 to run
9. Should break in main().
